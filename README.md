# Convercy
This application displays current exchange rates between different currencies and helps convert money from one currency to another.

### Technologies used:
- Kotlin
- MVI
- Kotlin Flow
- Jetpack Compose
- Coroutines
- Koin
- Retrofit
- Cbr-xml-daily API (https://www.cbr-xml-daily.ru)
